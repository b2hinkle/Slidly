Slidly

Slidly is the only parallax implementation with zero jitter and zero lag. Lightweight, zero dependencies, GPU-offloaded, and mobile-friendly.

View on GitHub

About

All animation/parallax libraries out there claim to be "smooth", but end up having some jitter/lag quirk with them. Slidly is different, as the whole motivation behind building it was solely to fix what other libraries failed at. Although perfect movement was the primary goal, workflow quality was also important to me so I ensured to make integration intuitive.
